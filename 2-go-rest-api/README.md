## commands
```bash
$ go mod init github.com/cesaralmeida93/learning-go-by-examples/2-go-rest-api
$ go run internal/main.go
$ curl localhost:8080
$ go build -o bin/2-go-rest-api internal/main.go
$ brew install go-task/tap/go-task
$ task --list
$ brew tap go-swagger/go-swagger
$ brew install go-swagger
$ swagger version
$ task swagger.validate
$ task swagger.doc     
```
